# 1. Introduction
Theano, Lasagne, Nolearn是各自相依的python library，用於deep learning。安裝時需要依照以下版本需求進行裝設以免相容失敗:

**Theano requires:**

* python 2.6 (or python 3.3)
* g++, python-dev
* Numpy 1.7.1 (or newer)
* Scipy 0.11 (or newer)
* A BLAS installation (with Level 3 functionality)

**Lasagne requires:**

* Theano-0.8.0

**Nolearn requires:**

* numpy-1.10.4
* scipy-0.16.1
* Theano-0.8.0
* Lasagne-0.2
* joblib-0.9.3
* scikit-learn-0.17
* tabulate-0.7.5
